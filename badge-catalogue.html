FIWARE Catalogue

Standardized FIWARE badges should be used to represent the various sub-chapters found within the FIWARE Catalogue. The larger, fully styled badge with the FIWARE logo in the correct font should be used whenever possible, but a simplified alternative is also available to match the smaller badge styling found on GitHub.

To download a badge select the preferred image format and click on the download button. Alternatively, you can also click on the copy icon to obtain the URL for the badge image hosted on our CDN server and use them within an tag. To obtain the URL for the simplified GitHub badges, click on the GitHub icons.
